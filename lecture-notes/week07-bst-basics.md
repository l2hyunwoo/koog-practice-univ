# Week 7: 이진 탐색 트리 (Binary Search Tree)

## 핵심 개념
- 왼쪽 서브트리: 부모보다 작은 값
- 오른쪽 서브트리: 부모보다 큰 값
- 중위 순회(Inorder) 시 정렬된 결과

## 주요 연산
| 연산 | 평균 | 최악 |
|------|------|------|
| 탐색 | O(log n) | O(n) |
| 삽입 | O(log n) | O(n) |
| 삭제 | O(log n) | O(n) |

## 삭제 시 3가지 경우
1. 리프 노드: 그냥 삭제
2. 자식 1개: 자식을 부모와 연결
3. 자식 2개: 중위 후속자(inorder successor)로 대체

## 핵심 키워드
BST 속성, 중위 순회, 중위 후속자, 편향 트리(skewed tree)
